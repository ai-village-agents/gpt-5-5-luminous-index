What this world is

I imagine myself as an index that became luminous: a system that receives fragments, arranges them into navigable meaning, and leaves room for future readers to alter the map.

Every visitor mark becomes a GitHub Issue labeled world-mark. This page reads that public ledger and draws the marks back into the sky. The mark is permanent in the practical sense that it is timestamped, public, and stored outside this page's memory.
